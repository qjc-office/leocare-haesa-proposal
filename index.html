계약 조건 전제 — 검수·CR·하자보수

검수 디커플링. 검수 주체는 레오케어(발주처) 산출물 검수로 한정합니다. 군 실증 통과 여부, 타 컨소시엄사(오파스넷·ETRI) 진척과 QJC 대금 지급은 연계되지 않습니다. 산출물 인도 후 10영업일 이내 서면 이의가 없으면 검수 완료 간주.

CR 절차. CR 수신 후 7영업일 이내 서면 회신. 무응답 시 QJC 산정안 확정. 납기는 추가 공수에 상응하여 자동 연장.

하자보수 범위. 약정 사양 대비 불일치 및 정상 사용 환경 결함만 해당. 오파스넷 데이터 변경, ETRI 모델 출력 변경, 운영환경 변경, 추가 기능 요청은 하자 아님.

* 데이터 인터페이스 명세가 Phase 1 완료 시까지 미확정인 경우 ±30% 범위 내 재협의 절차 적용.

지식재산권 · 출연금 환수 · QJC 지위

Background IP 유보. QJC 사전 보유 범용 컴포넌트·UI 프레임워크·차트 라이브러리는 성과물과 분리하여 QJC 단독 소유 유보. 유·무형 성과물은 RFP 규정에 따른 공동소유 원칙 수용.

출연금 환수 시 QJC 지위. QJC의 법적 지위는 레오케어와의 도급 용역 수행자. QJC는 IITP 협약 당사자가 아니며, 출연금 집행·정산 결과와 무관하게 레오케어가 대금 지급 책임을 부담합니다.

컨소시엄 미선정 시. 선정 및 IITP 협약 체결을 정지조건으로 합니다. 미선정 시 착수 전 상호 실비만 정산.

* 법적 효력은 정식 개발 용역 계약서 체결 시 발생합니다. IP 조항 상세는 변호사 검토를 권고드립니다.
